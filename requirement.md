# 北台灣一日遊

## Description
因為假日出去玩常常不知道要去哪邊走走，如果都去逛商場又一直花錢不太健康，故需要製作一個抽籤程式，可以抽出週末出去玩的地點，並且可以設定是否要找免費、免門票景點等。

## Goal
- 在 google search 上面爬蟲, 可以透過 map 等工具找到北台灣適合一日遊的景點。
- 景點的定義
  - 自然景觀步道
  - 大型購物商場
  - 遊樂場
  - 博物館
  - 大型觀光工廠
  - 動物園
  - 大型公園
  - 觀光農場
- 北台灣的定義, 以下縣市
  - 基隆
  - 台北
  - 新北
  - 桃園
  - 新竹
  - 苗栗
- 景點需要被分類, 建議使用 CSV 作為資料庫存取的格式, 需要分類的項目有:
  - 景點名稱
  - 縣市
  - 是否為付費景點
    - 免費
    - 內部消費
    - 門票
  - 特別適合年齡層
    - 都適合
    - 小孩
    - 中年人
    - 老年人
  - Google map 連結
  - 參考資料 (請蒐集網友們的blog或者官方網站、社群媒體等)
- 抽籤程式
  - 需要有界面
  - 需要有filter
    - 地區
    - 費用
    - 年齡層
  - 請使用以下語言各實做一個版本
    - Shell: 
      - 純文字輸入輸出, 使用argv作為input控制
      - 具有LOG功能, 也是使用CSV進行控制
    - python:
      - 需要有UI界面給使用者操作
      - 具有類似拉霸的動畫或者快速跑馬燈讓使用者以為真的有在抽籤
      - 具有LOG功能, 可以看到以前抽到的位置
